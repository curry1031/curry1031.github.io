    can通信编程

    在linux系统中会有can0（实际的物理连接就是外接硬件设备如PEAK-can）网卡，可以使用socketcan编程

    1.加载CAN总线驱动程序 sudo modprobe can

    2.创建CAN通道 sudo ip link set can0 type can bitrate 500000

    3.验证CAN通道 ifconfig

    SocketCAN对全平台支持都很好，除了Windows的丰富资料外，在Linux端只需安装net-tools和SocketCAN驱动，然后就可以使用utils的命令去控制。

    除了物理连接还有虚拟的网口，是在内存中模拟的,不过这无法接触到真实的can数据

    1.加载CAN总线驱动程序 sudo modprobe vcan

    2.创建CAN通道 sudo ip link add dev vcan0 type vcan;sudo ip link set up vcan0

    如果想要跨平台进行can通信可以使用python-can库或是usb-can进行上位机的开发
